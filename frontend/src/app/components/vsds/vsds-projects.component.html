A sheet variant defines how the system recognises and reads a specific survey spreadsheet format. When a spreadsheet is processed, each variant is tested in order; the first one whose header checks all pass is used to extract the data.

Each variant specifies:

  • Header Row — the row containing column titles (data starts on the next row).
  • Four column positions — system name, Z-sample, system count, and max distance.
  • Header Checks — cell values that must all match for this variant to be selected.

Columns use spreadsheet letters (A, B … Z, AA …); rows are 1-indexed. The system stores them as 0-indexed — conversion is automatic.
